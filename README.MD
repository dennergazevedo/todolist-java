# To-Do List

Backend de um To-Do list apenas para estudo, fortalececimento e expansão dos conhecimentos da Stack **Spring Boot** + **Hibernate** + **H2DB**

## Start Project
Execute `mvn spring-boot:run` no terminal da pasta para executar o projeto.

## Métodos
### user
 - [POST] create: `/user/`
### task
- [POST] create: `/task/`
- [GET] list: `/task/`
- [PUT] update: `/task/{id}`
